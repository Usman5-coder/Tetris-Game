## 🕹️ How to Play

- ⬅️ / ➡️ – Move piece left/right  
- ⬇️ – Soft drop  
- ⬆️ – Rotate  
- `Space` – Hard drop  
- `P` – Pause/Resume  

Clear rows to gain points. The more rows you clear at once, the higher the score!
